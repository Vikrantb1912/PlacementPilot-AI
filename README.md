# PlacementPilot AI

## Overview

PlacementPilot AI is an intelligent Agentic AI assistant built using IBM watsonx Orchestrate.

It helps engineering students prepare for internships and campus placements by providing personalized study plans, mock interviews, resume improvement suggestions, project recommendations, and career guidance.

---

## Problem Statement

Many engineering students lack structured guidance for campus placement preparation. They often struggle to decide what to study, which skills to learn, and how to prepare for interviews.

---

## Proposed Solution

PlacementPilot AI acts as a virtual placement mentor that interacts with students, understands their profile, and provides customized guidance based on their goals and skill level.

---

## Features

- Personalized placement roadmap
- Mock HR interviews
- Technical interview preparation
- Resume improvement suggestions
- Project recommendations
- Skill recommendations
- Career mentoring

---

## Technologies Used

- IBM watsonx Orchestrate
- Large Language Model (GPT-OSS 120B)
- Prompt Engineering
- Agentic AI

---

## Project Workflow

1. Student enters career goals.
2. Agent asks follow-up questions.
3. Agent analyzes the student's profile.
4. Agent generates personalized recommendations.
5. Student interacts continuously for placement preparation.
